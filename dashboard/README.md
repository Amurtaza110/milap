# Milap Admin Dashboard

A role-based web dashboard for managing the Milap ecosystem.

## Roles & Features
- **Customer Support**: Ticket management, User inquiries.
- **Connect Manager**: Approvals (Milap+), Payments, Security.
- **Marketing**: Push notifications, Campaigns, Analytics.
- **Developer**: Logs, Feature flags, Database view.
- **Owner**: Full access + Financials.

## Tech Stack
- **Frontend**: React (Vite) + Tailwind CSS
- **Backend**: Node.js (Express) [Mocked/Integrated]
- **Auth**: JWT + Role-based Guards

## Setup
1. `npm install`
2. `npm run dev`
